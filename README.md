# 🩺 Health AI — AI-Powered Disease Symptom Checker

A full-stack Gen AI project that uses RAG (Retrieval Augmented Generation) 
pipeline + Gemini AI to diagnose diseases based on symptoms.

## 🚀 Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js + Express
- **AI Engine:** Python + FAISS + Gemini API
- **Dataset:** 41 diseases, 132 symptoms (Kaggle)

## ✨ Features
- RAG Pipeline with FAISS vector search
- Gemini AI diagnosis
- Sample symptom chips
- Recent search history
- Analytics dashboard
